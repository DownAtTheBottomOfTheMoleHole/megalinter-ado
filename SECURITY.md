# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately using GitHub's Security Advisories feature:

**[Report a vulnerability](https://github.com/DownAtTheBottomOfTheMoleHole/megalinter-ado/security/advisories)**

Do **not** disclose security issues publicly until they have been reviewed and patched.

- We will respond as quickly as possible.
- After the vulnerability is addressed, we will disclose the issue and fix in the [CHANGELOG.md](CHANGELOG.md).
