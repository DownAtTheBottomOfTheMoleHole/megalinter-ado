# MegaLinter Flavors

MegaLinter offers optimized flavors for different tech stacks. Using a flavor reduces Docker image size and pull time.

| Flavor        | Description                         |
|---------------|-------------------------------------|
| all           | All available linters               |
| ci_light      | Lightweight CI optimized            |
| cupcake       | Popular linters for common projects |
| documentation | Documentation files only            |
| dotnet        | .NET projects                       |
| dotnetweb     | .NET web projects                   |
| formatters    | Code formatters only                |
| go            | Go projects                         |
| java          | Java projects                       |
| javascript    | JavaScript/TypeScript projects      |
| php           | PHP projects                        |
| python        | Python projects                     |
| ruby          | Ruby projects                       |
| rust          | Rust projects                       |
| salesforce    | Salesforce projects                 |
| security      | Security-focused linters            |
| swift         | Swift projects                      |
| terraform     | Terraform/IaC projects              |

See [MegaLinter Flavors](https://megalinter.io/latest/flavors/) for full details.
