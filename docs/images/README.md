# Screenshot Placeholders

Add the following screenshots to this folder:

## Required Screenshots

1. **task-config.png** - Screenshot of the MegaLinter task configuration UI in Azure DevOps
   - Show the task assistant with all input fields visible
   - Highlight key options like flavor, release, fix, enablePRComments

2. **pipeline-run.png** (optional) - Screenshot of a successful pipeline run
   - Show the MegaLinter step output

3. **pr-comment.png** (optional) - Screenshot of PR comment output
   - Show inline linting feedback on a pull request

## How to Capture Screenshots

1. Open your pipeline in Azure DevOps: https://dev.azure.com/rolf-moleman/test/_build?definitionId=1
2. Edit the pipeline and click on the MegaLinter task
3. Capture the task configuration panel
4. Save as `task-config.png` in this folder

## Recommended Image Sizes

- Width: 800-1200px
- Format: PNG
- Compress if larger than 500KB
